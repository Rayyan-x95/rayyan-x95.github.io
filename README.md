# 🧠 Rayyan's Portfolio v1.4.2

> Designed, developed & deployed with love and a dash of Gen Z chaos ⚡ by [Rayyan](https://github.com/Rayyan-x95)

## 🎨 Overview

This is a fully responsive, single-file, dark/light-themed portfolio site created using HTML, CSS, and embedded JavaScript. It’s got flip card animations, glowing hover effects, SVG-enhanced iconography, and switches styles with the flick of a theme toggle. No frameworks. No clutter. Just clean vibes and clever UX.

## 🌟 Features

- ⚡ **Animated Skill Cards** — Flip on hover with smooth transitions
- 🌘 **Dark/Light Mode Toggle** — Icon + theme toggle, saved in `localStorage`
- 💡 **Glowing Hover Effects** — Subtle but noticeable, especially in dark mode
- 📱 **Fully Responsive** — Optimized for mobile, tablet, and desktop
- 🧩 **Single HTML File** — Clean, minimal, and ready to deploy
- 🎨 **Icon-Enhanced Skills** — With explanatory tooltips & flipbacks
- 🌐 **Language Switcher** *(JP/EN)* — Built-in support for future localization

## 📁 Structure

All code is contained in a single `index.html` file.
